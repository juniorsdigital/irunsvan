# IRUNSVAN Website Prototype

A full website prototype for IRUNSVAN — a carbon-plated performance running shoe brand.

## Live Preview

Open `index.html` in any browser. No dependencies or build steps required.

## Features

- 🟢 On-brand electric green (`#7FFF00`) color system on dark backgrounds
- Fixed navigation with blur backdrop and active scroll highlighting
- Promo banner with discount code
- Full-screen hero section with stats
- Animated marquee ticker
- Best Sellers product grid with hover effects and sale tags
- Category cards with cinematic overlay reveals
- Technology / features section with icon list
- Trust bar (Free Shipping, 20-Day Warranty, 24/7 Support)
- Customer reviews with star ratings
- FAQ accordion
- Blog strip (3 articles)
- Full-width CTA band
- Complete footer with nav columns and social links
- Scroll-reveal animations throughout
- Fully responsive (mobile, tablet, desktop)

## Stack

Pure HTML / CSS / JavaScript — zero dependencies.

## Brand Colors

| Token | Hex | Usage |
|---|---|---|
| `--green` | `#7FFF00` | Primary accent — CTAs, prices, highlights |
| `--green-dim` | `#5DCC00` | Hover state for green elements |
| `--dark` | `#0A0A0A` | Main background |
| `--mid` | `#111111` | Alternate section background |
| `--card` | `#161616` | Card backgrounds |
| `--orange` | `#FF6B00` | Secondary accent |
| `--border` | `#242424` | Borders and dividers |
